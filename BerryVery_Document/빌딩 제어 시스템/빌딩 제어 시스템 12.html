화면 구성 요소
1

서버 선택 (Multi-Server)

메인 서버 장애 시 백업 서버로 즉시 전환하거나, 테스트 환경으로 접속할 수 있는 드롭다운 메뉴.

2

보안 로그인 (MFA)

기본 ID/PW 인증 외에 외부 접속 또는 관리자 계정 접근 시 OTP 코드를 추가로 요구하는 MFA 필드.

3

자동 로그인 및 옵션

공용 PC가 아닌 경우 재접속 편의를 위한 자동 로그인 옵션과 비밀번호 분실 시 복구 링크 제공.

4

시스템 상태 바

로그인 시도 전 서버와의 통신 연결 상태(Ping)와 클라이언트 버전을 하단에 표시하여 문제 진단 용이.

핵심 시나리오

#1. 최초 로그인 및 서버 설정

'Main Server' 선택 후 연결 상태 'Online' 확인
관리자 계정 입력 후 로그인 버튼 클릭

#2. MFA 추가 인증 (보안 강화)
