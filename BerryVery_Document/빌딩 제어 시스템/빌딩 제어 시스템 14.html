화면 구성 요소
1

단계별 진행바 (Stepper)

복잡한 등록 절차를 4단계(기본정보, 권한, 서류, 완료)로 나누어 사용자의 입력 부담을 줄이고 진행 상황을 시각화.

2

객실 배정 및 기간 설정

거주자/투숙객 유형에 따라 공실을 조회하여 배정하고, 입주 기간을 설정하여 권한 유효 기간과 연동.

3

출입 권한 프리셋

선택한 객실 및 사용자 유형에 맞춰 기본 출입 구역(현관, 주차장 등)이 자동 체크되며, 추가 시설 권한을 선택 가능.

4

카드/QR 미리보기

발급될 모바일 출입증의 디자인과 정보를 실시간으로 미리 보여주어 오발급을 방지.

핵심 시나리오

#1. 신규 투숙객 체크인

투숙 기간(Check-in/out) 설정 시 권한 자동 만료 예약
'모바일 QR' 발급 선택 후 고객 스마트폰으로 전송

#2. 객실 재배정 (권한 이관)

기존 1005호 -> 1201호 변경 시 출입 권한 자동 업데이트
